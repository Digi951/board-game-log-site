Support

Danke, dass du BoardGameLog verwendest. Wenn du Fragen hast, einen Fehler melden möchtest oder Feedback zur App geben willst, freue ich mich über eine Nachricht.

BoardGameLog speichert alle Daten lokal auf deinem Gerät. Es gibt in der aktuellen Version keine automatische iCloud-Synchronisierung. Deine Daten lassen sich jederzeit über Einstellungen → Export & Import als JSON sichern und wiederherstellen.
